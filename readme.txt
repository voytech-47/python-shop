1. Pobrać Python 3.11 z https://www.python.org/ftp/python/3.11.5/python-3.11.5-amd64.exe (Windows, 64-bit)
2. Uruchomić terminal (najlepiej Powershell)
3. py -m ensurepip --upgrade
4. pip -r requirements.txt (musisz być w odpowiedniej ścieżce albo podać ścieżkę absolutną pliku requirements)
5. Pobrać i zainstalować XAMPP
6. Uruchomić XAMPP i włączyć usługi Apache i MySQL
7. w przeglądarce wejść na localhost/phpmyadmin
8. Do konsoli (na dole) wkleić całą zawartość pliku queries.txt
9. python app.py (w terminalu Powershell, musisz być w odpowiedniej ścieżce albo podać ścieżkę absolutną pliku app.py)
10. w przeglądarce localhost:5000, aby przejść do aplikacji
