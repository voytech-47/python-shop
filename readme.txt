1. Pobrać Python 3.11 z Microsoft Store
2. Uruchomić terminal (najlepiej Powershell)
3. pip -r requirements.txt (musisz być w odpowiedniej ścieżce albo podać ścieżkę absolutną pliku requirements)
4. Pobrać i zainstalować XAMPP
5. Uruchomić XAMPP i włączyć usługi Apache i MySQL
6. w przeglądarce wejść na localhost/phpmyadmin
7. Do konsoli (na dole) wkleić całą zawartość pliku queries.txt
8. python run.py (w terminalu Powershell, musisz być w odpowiedniej ścieżce albo podać ścieżkę absolutną pliku app.py)
9. w przeglądarce localhost:5000, aby przejść do aplikacji
